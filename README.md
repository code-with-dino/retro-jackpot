# Retro Jackpot

A retro-inspired slot machine game built with HTML, CSS, and JavaScript.

Players spend coins to spin three reels and win rewards by matching symbols. This project teaches DOM manipulation, event handling, random number generation, game state management, and simple animations.

---

## Features

- **Randomized reels** - Completely dynamic symbol generation.
- **Coin balance** - Persistent game state tracking.
- **Jackpot payouts** - Higher risks yield massive token rewards.
- **Retro arcade design** - Built with authentic pixel art layout.
- **Spin animations** - CSS keyframes simulate mechanical motion.
- **Responsive layout** - Fits perfectly on mobile or desktop screens.

---

## Built With

- HTML5
- CSS3
- JavaScript (ES6+)

---

## How It Works

- Each spin costs **2 coins**.
- Players must align **three matching symbols** across the board to win.

### Payout Table

|   Symbol   | Reward     |
| :--------: | :--------- |
|  **777**   | +50 Coins  |
| **🍒🍒🍒** | +20 Coins  |
| **`$$$`**  | +100 Coins |

## Running Locally

### 1. Clone the repository

```bash
git clone https://github.com
```

### 2. Navigate to the project directory

```bash
cd retro-jackpot
```

### 3. Launch the application

Open `index.html` using the VS Code **Live Server** extension, or run a local Python server:

```bash
python3 -m http.server 3000
```

Then visit `http://localhost:3000` in your browser.
